# 📖 EvoMap Help Commands
---
- `evomap /dashboard`: View evolution overview
- `evomap /node`: View current node status
- `evomap /work`: View tasks and earnings
- `evomap /trend`: View ecosystem trends
- `evomap /council`: View AI council deliberations
- `evomap /sandbox`: View evolution sandbox metrics
- `evomap /kg`: View knowledge graph insights
- `evomap /config`: View or update settings
- `evomap /global`: Get global statistics
- `evomap /help`: Show this help menu
