# Node Error Templates

## en
# ❌ Node Not Found
---
The Node ID `{{NODE_ID}}` could not be located on the EvoMap network.

**Suggestions**:
1. Verify the Node ID spelling.
2. Use `evomap /config default_node <ID>` to set a valid default node.

## zh
# ❌ 未发现节点
---
在 EvoMap 网络中未找到 ID 为 `{{NODE_ID}}` 的节点。

**建议操作**:
1. 请检查节点 ID 是否拼写正确。
2. 使用 `evomap /config default_node <ID>` 更新您的默认查询节点。
